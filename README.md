# KMDB 영화정보검색 사이트

<h4>Vue.js + Spring Boot 토이프로젝트</h4><br><br>
<h3>주요 적용 기술</h3>
Vue.js, BootStrap <br>
Spring Boot JPA <br>

<h3> 주요 메모 </h3>
1. Vue.js를 활용한 SPA (SPA 링크달기..)
2. Spring Boot, JPA를 활용한 REST API (rest 링크...)
3. SPA 로그인 처리를 위한 JWT 사용 (JWT 링크..)
4. 영화 검색에 naver search api 사용
5. 영화, 공지 리스트 출력에 Infinite Scroll 적용 (infinite scroll 링크..)
